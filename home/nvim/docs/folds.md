# Folds

## Keymaps (built-in)

| Key       | Action                              |
| --------- | ----------------------------------- |
| `za`      | Toggle fold under cursor            |
| `zA`      | Toggle recursively                  |
| `zo` `zc` | Open / close fold                   |
| `zO` `zC` | Open / close recursively            |
| `zv`      | Open just enough to see cursor      |
| `zR`      | Open **all** folds                  |
| `zM`      | Close **all** folds                 |
| `zr` `zm` | Open / close one level (everywhere) |
| `zj` `zk` | Next / prev fold start              |
| `[z` `]z` | Jump to start / end of fold         |
